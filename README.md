## 🚀 Capstone Projects

### 🍋 Little Lemon — React Frontend
**Course:** Front-End Developer Capstone
A responsive restaurant web app built with React. Features a table booking system with form validation, date/time selection, and a confirmed booking screen.
→ [`frontend/capstone/`](./frontend/capstone/README.md)

### 🍋 Little Lemon — Django Backend
**Course:** Back-End Developer Capstone
A Django REST API for the Little Lemon restaurant. Includes menu browsing, table booking, user authentication, and a fully tested API layer.
→ [`backend/capstone/`](./backend/capstone/README.md)
